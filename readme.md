# React UI Kit Demo

An example UI Kit for react with self-documenting components.

## Dev
run `gulp`

# Generate Docs
run `react-docgen ./ -o ./ComponentDocs.json`

## Tests
run `gulp test`
 https://blog.algolia.com/how-we-unit-test-react-components-using-expect-jsx/
